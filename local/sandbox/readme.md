# Sandbox

Папка `sandbox` используется как локальная учебная и тестовая зона внутри `local` в коробке Bitrix24.

## Назначение

Здесь размещаются небольшие страницы, примеры, черновики и экспериментальные заготовки, которые нужны для изучения возможностей Bitrix24 и проверки отдельных механизмов в работе.

Внутри папки можно:

- тестировать API и D7-подходы;
- проверять работу инфоблоков, ORM, компонентов, событий и обработчиков;
- делать временные страницы для отладки;
- собирать маленькие POC-сценарии;
- изолированно изучать поведение Bitrix без вмешательства в основную проектную логику.

## Что важно

Папка `sandbox` не предназначена для production-кода.  
Это рабочая песочница для экспериментов, обучения и технических проверок.

Рекомендуется держать здесь:

- только понятные и локализованные тесты;
- раздельные подпапки по темам;
- минимально необходимый код;
- комментарии, если тест неочевидный.
